PowerShell can't be exempted

Script hosts run other people's code. An exemption here would silently apply to every script PowerShell runs — signed or not — so interpreters can never be exempted, regardless of signature.

This isn't configurable. If a specific workflow is noisy, exempt the signed application that drives it instead.
